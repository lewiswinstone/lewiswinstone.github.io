Pattern Libraries

A pattern is a reusable solution to a problem that often occurs in UX and interaction design. These solutions can vary from different elements such as buttons, navigation, grid layouts and many others. A collection of these user interface elements are referred to as a patter library.

Within this pattern library the componenets that will be covered are;
• Navigation Bar
• Header Carousel
• Buttons
• Jumbotron
• Footer
• Grid Layouts

Each component will be explained and supported by a working example.
